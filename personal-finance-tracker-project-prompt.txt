
Project Title: Personal Finance Tracker with Charts

üí° Overview:
Build a responsive, animated Personal Finance Tracker web app using HTML, CSS, and JavaScript with local data persistence. The app should allow users to track income and expenses, categorize transactions, and visualize data with charts. The design should be sleek, modern, and responsive, with hover effects, smooth animations, and a loader on launch.

üéØ Objectives:
- Add income and expense transactions
- Categorize each transaction (e.g., Food, Rent, Salary, Entertainment)
- Display a summary (total income, expenses, and balance)
- Use localStorage for data persistence
- Visualize data using Chart.js (pie chart, bar chart)
- Responsive layout with animations and hover effects
- Styled with a modern, minimal theme and top-tier fonts

üîß Technologies:
- HTML5
- CSS3 (Flexbox/Grid, transitions, animations)
- Vanilla JavaScript (ES6+)
- Chart.js for data visualization
- localStorage API

üß† Features:

1. **Add Transaction**
   - Input: Title, Amount, Type (Income/Expense), Category
   - Validate inputs before adding
   - Reset form with animation

2. **Transaction List**
   - Display transaction cards with icons, title, amount, and category color
   - Smooth fade-in on add, fade-out on delete
   - Hover effect: highlight card border or show delete/edit icon

3. **Financial Summary**
   - Display total income, total expenses, and remaining balance
   - Use color-coded indicators

4. **Charts & Analytics**
   - Pie chart: Expense distribution by category
   - Bar chart: Income vs Expenses per month
   - Auto-update charts on data change

5. **Persistent Storage**
   - Store all transactions in localStorage
   - Load data on page reload

6. **Loader Animation**
   - Loader spinner or branded logo animation while loading

7. **Animations & Hover Effects**
   - Animate card entries, modals, and buttons
   - Add hover effect for interactions using scale or shadow

8. **Fonts & Styling**
   - Use "Poppins" or "Inter" font from Google Fonts
   - Font sizes:
     - 24px for headers
     - 16px for labels/text
     - 14px for meta info
   - Consistent spacing and color balance
